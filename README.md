# FoodOrderingSystem_Sid

Description: A online system developed for addering issue of wrong order placing and to provide seamless payment method to the customers. the development platform used is netbeans 8.2 and technologies used are java and mysql. the system provide easy user interface, multiple cafes or restaurants options, and multiple payment methods.

System flow: user -> register/ login -> browse cafe or restaurant -> browse menu -> place order -> make payment

Status: main modules are fully functional and for online payment only demo implementation is done for security purpose.
